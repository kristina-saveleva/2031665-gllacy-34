# Личный проект «Глейси»
Верстка сделана по дизайну HTML-academy.
Работа с JavaScript выполнена полностью самостоятельно. Придумывала, какие функции хочу добавить в работу и добавляла сама.

Сделаны самостоятельно: 
* Создание карточек при открытии каталога (функция renderCard),
* SlickSlider на главной странице, а также на карточке товара,
* Открытие карточки товара

Планируется доделать:
* Поиск по сайту,
* Фильтры товаров,
* Функция добавления/удаления товара из корзины


Работа над страницей продолжается.
